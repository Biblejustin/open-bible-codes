# Notice

Open Bible Codes is a from-scratch implementation.

Bible source texts are not vendored into this repository. Download/import scripts keep
raw and processed corpora under ignored `data/raw/` and `data/processed/` paths so
each corpus can keep its own source notice and license metadata.

No source code is copied or vendored from:

- TorahBibleCodes
- bible-codes.github.io
- torahcodespython
- els-engine
- other ELS projects

Existing projects may be used only for feature comparison, terminology checks, or independent result comparison.

Raw Bible texts are not included. Users must provide texts with suitable rights for their use case.
