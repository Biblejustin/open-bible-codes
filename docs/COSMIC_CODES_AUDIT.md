# Cosmic Codes Source Audit

Source: Chuck Missler, `Cosmic Codes: Hidden Messages from the Edge of
Eternity`, Koinonia House, revised 2004, third printing 2011. Reviewed from
the user-supplied PDF `Cosmic_Codes_eBook.pdf`; the PDF is copyrighted and is
not committed to this repository.

Status: source audit and screening-list expansion. This is not a reproduction
report.

## Why This Source Matters

`Cosmic Codes` is a broad popular-level Bible-code treatment that combines:

- common Torah/YHWH skip examples;
- historical ELS claims attributed to Witztum/Rips/Rosenberg, Michelson,
  Satinover, Gans, Jeffrey, and Rambsel;
- source-sensitivity statements about Koren Jewish Torah text, BHS differences,
  and letter-level Torah variation;
- explicit negative-control challenges from `Moby Dick`, `War and Peace`, and
  software-license examples;
- Yeshua/Messiah examples in Genesis, Ruth, Daniel, Zechariah, and Isaiah 53;
- an Isaiah 52:13-53:12 table with named people, places, events, and passion
  terms;
- a Holocaust-code table with spellings, intervals, and starting passages;
- broader non-ELS "macrocodes" including heptadic word-count claims, gematria,
  acrostics, notarikon, and rhetorical structures;
- a strong warning that ELS findings should not be used for divination,
  future prediction, doctrine, or theology.

## Methodology Notes

This source is useful because it states several rules we should preserve in the
research pipeline:

- single ELS words are weak without relevant passage context, clustering, or
  explicit controls;
- source editions matter at letter level, especially Koren-style Torah versus
  critical Hebrew editions;
- multiple-testing and post-selection are central risks;
- non-Bible texts must be checked rather than dismissed;
- hidden-path findings may authenticate only what is already known from plain
  text and should not be used to create doctrine;
- exact tables need exact source text, row width, start letter, direction,
  skip, and normalization before reproduction language is allowed.

## Reproduction Requirements

Before elevating any row:

1. lock the source text edition and normalization;
2. transcribe the exact Hebrew, Greek, or English target string;
3. lock book, passage, start word, start letter, skip, direction, and row width;
4. preserve any mismatch between Missler, Jeffrey, Rambsel, or later audit
   spellings;
5. distinguish ELS claims from acrostic, notarikon, gematria, word-count, and
   macro-structure claims;
6. run Bible-source distribution checks and matched non-Bible controls;
7. correct for the full declared family of tested terms and layouts.

## Cautions

- This is a copyrighted secondary source, not redistributable source data.
- Some Hebrew strings in the PDF extract are transliterated or visually noisy;
  rows marked provisional need a line-by-line source audit before reproduction.
- Several claims are broad clusters rather than exact letter-path claims.
- The book itself warns against using ELS codes for prediction, divination, or
  doctrinal discovery.
- These additions expand the audit surface; they do not validate the claims.
